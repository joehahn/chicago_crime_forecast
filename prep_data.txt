The following will prepare the Chicago crime data for training ML models to orecast crimes across that city.
Execute the following completely from scratch. 
Do not use any cached results, temporary files, or previously computed outputs. 
Do not gather any files using git
Clear any cache files first, then execute everything fresh.

Begin by loading file data/crimes.csv into object df_filtered.

Profile df_filtered.

Start with df_filtered and rename the 'date' column as 'timestamp'.
And derive the month of the year from the timestamp column and name that new column 'month'.
Then preserve only those records whose primary_type are among the top 20.
Store results in object df_20. How many records are in df_20?

Display counts of primary_type in df_20

Display 1 random record in df_20, show all columns

What are the column types in df_20?

Display minumum date and maximum date

Group df_20 by year, month, ward, primary_type and calculate 
    first date of the month(timestamp) as 'date',  
    mean(arrest), 
    mean(domestic), 
    mean(latitude), 
    mean(longitude), 
    count(id) as 'count_0', 
Drop 'min_', 'max_, 'mean_' from all new columns. 
Cast ward column as integer. 
Reorder columns so that date is first and count_0 is last.
Order all records by date, ward, primary_type.
Name this new result df_avg. How many records are in df_avg?

Which ward in df_avg has the greatest sum(count_0) among primary_type=THEFT records?
Show all records in df_avg having primary_type=THEFT and ward=42

Show all records in df_avg having primary_type=ARSON and ward=42

Use the year, month, ward, primary_type columns to determine which records are missing from df_avg. 
Zero-pad those missing records in df_avg with count_0 set to 0.
Be sure to also include the missing values in the date, years, and month columns.
Order all records by date, ward, primary_type.
Store that result in object df_pad.
How many records are in df_pad?

Show all records in df_pad having primary_type=ARSON and ward=42

For each column in df_pad, find any NaN values and replace them with random selections of non-NaN values.
Store that result in object df_nan.

Show all records in df_nan having primary_type=ARSON and ward=22

Print 5 random records in df_null

Partition df_null by year, month, ward, primary_type and order by date.
Append a new column count_previous which is count_0 shifted backwards in time by 1 month.
Append a new column count_1 which is count_0 shifted forwards in time by 1 month.
Also append a new column count_2 which is count_0 shifted forwards in time by 2 months.
And append another new column count_3 which is count_0 shifted forwards in time by 3 months.
And append another new column count_4 which is count_0 shifted forwards in time by 4 months.
Set delta_count = count_0 - count_previous
Do not drop any records due to missing data, instead use NaN to indicate any missing future data.
Also reorder the columns in with date first and delta_count, count_0, count_1, count_2, count_3, count_4 being last. 
Name the resulting object df_target

Print all records in df_target that have primary_type=THEFT and ward=27

Append a new column to df_target called 'ran_num' that has random floating point values 
uniformly distributed between 0 and 1, name that result df_tt.
The create a new column 'TTVFI' in df_tt
such that all rows in df_tt having ran_num <= 0.667 have TTVFI ='train',
          all rows in df_tt having ran_num  > 0.667 have TTVFI ='test'.

Set df_ttv=df_tt
and set last_few_dates=the 6 greatest dates in df_ttv,
and set column TTVFI ='validate' in df_ttv for all records having date in last_few_dates.

Set set df_ttvf=df_ttv
and set last_few_dates=the 2 greatest dates in df_ttvf,
and set column TTVFI ='forecast' in df_ttvf for all records having date in last_few_dates.

Set set df_ttvfi=df_ttvf
and set last_few_dates=the greatest date in df_ttvfi,
and set column TTVFI='ignore' in df_ttvfi for all records having date in last_few_dates.

Then drop these columns from df_ttvfi: arrest, domestic , count_previous, ran_num
Name that new result dt_monthly.

Drop from dt_monthly all records having delta_count=NaN

Prettyprint ALL records in dt_monthly having primary_type=THEFT and ward=22, and dont skip any records.

Save dt_monthly as file data/crimes_monthly.csv
